# Changelog

All notable changes to this project are recorded here. The project is still pre-release, so entries
are grouped by development milestones rather than public version tags.

## 0.1.0 - Development-Ready UI Repo

- Added the Arcade UI companion-client foundation for `Warhammer_40k_AI`.
- Added strict UI/core client facade types, local-session smoke support, and deterministic fake
  clients for UI tests.
- Added finite decision submission, movement path drafting, per-model movement assignments, and
  movement proposal submission diagnostics.
- Added live-core smoke coverage for real movement decisions and projection parsing.
- Added shareable YAML/JSON UI preferences, HUD zone layout profiles, HUD widget preview, ergonomic
  HUD summaries, generic assignment HUD, and advisory action visual summaries.
- Added forensic event tracing, crash diagnostic bundles, GUI event driver tests, and headless render
  evidence capture.
- Added CI, coverage thresholding, import-boundary auditing, regression fixtures, ADRs, and packaging
  smoke coverage.
